互联网大厂纷纷裁员，未毕业的你是否做好了顺利就业的准备？

p { color: red }

一

自一月疫情的影响，使得各个行业的发展都不景气。互联网行业尤为严重，不论是还在刚起步的科技公司还是BATJ这样的大厂，都有了相应的人员缩减和招聘缩减（大家对比去年的或者前年的招聘职位数量和今年的就可以看到）。

就在4月18日，微博爆出了蘑菇街裁员的消息，而且裁员比例竟然高达百分之14%，对于员工人数在1000+的互联网公司。也就意味着将近150多人面临离职的危险。难道真的像有些人说的：2019年是过去十年互联网行业最不景气的一年，但有可能是未来十年最景气的十年？

二

水涨船高，反之亦然，2020年，在这个特殊的时期在这个行业摸爬滚打十多年的老司机都有可能面临着辞退的危险。我们这些还未出校门的新手怎么能没有危机感。

互联网是不相信眼泪的，同样每一个行业都是不相信眼泪的。裁员确实危机，但是仔细想想也还好，物竞天择，适者生存这是亘古不变变得真理。企业受到了影响，无法养活更多的员工，那么就需要裁减一部分人员来减小开支，人之常情，如果我们作为企业的高管，在企业面临这样的危险的时候我想也会是同样的选择。但是裁员也不是随机裁的吧，我们要知道你的价值和你为公司带来的利润是和你的裁员风险成反比的，换句话说，留下的是相比其他成员能够给公司带来效益的人，走掉的是公司认为在应该的工薪分为内，没有给公司创造相应利益的人。现在的社会就是这样。我想说道这里，大家应该明白了吧：不论是在校还是在企业，夯实的基础+强硬的专业技能+永不停息学习的脚步，跟上计算机行业的脚步，是你立于不败之地的定海神针

行业是不景气，对于应届生还有未毕业的人都会有影响。但是即使企业缩招，还是要招人的，只不过岗位数量和要求有所提升。如果你在同一批次的求职者中拥有夯实的基础+强硬的专业技能，同时又善于学习，坚持学习，那么是不是比其他人更有优势，我想无论是哪个企业都愿意招有能力，有价值，能够创造效益的员工吧。

坚持学习，洞察前沿，了解市场，让技术服务于市场，时刻的危机便是前进的动力，加油！

互联网不相信眼泪，但是它相信技术!
